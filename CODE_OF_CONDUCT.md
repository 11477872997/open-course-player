# 行为准则

本项目遵循简单规则：

- 尊重他人。
- 直接、友善地讨论问题。
- 默认善意，但及时处理伤害性行为。
- 议题和合并请求聚焦项目本身。
- 不利用本项目请求或传播非法版权内容。

维护者可以移除辱骂、骚扰、垃圾信息、不安全或与项目无关的内容。

---

# English

This project follows a simple standard:

- Be respectful.
- Discuss ideas directly and kindly.
- Keep issues and merge requests focused on the project.
- Do not use this project to request or distribute illegal copyrighted content.

Maintainers may remove abusive, harassing, spammy, unsafe, or unrelated content.
