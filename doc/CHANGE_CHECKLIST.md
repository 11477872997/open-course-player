# 变更检查清单

每次变更完成前都按这个清单过一遍。

## 范围

- 是否影响播放行为？
- 是否影响支持格式？
- 是否影响本地文件访问？
- 是否影响安装包内容？
- 是否新增或删除依赖？
- 是否影响隐私、日志或网络访问？

## 前端

- 目录树是否仍能处理长中文和英文文件名？
- 空状态、加载中、错误和不支持格式状态是否完整？
- 最小窗口尺寸下播放器控制条是否可用？
- 如果新增快捷键，是否补了文档？
- 切换文件时是否销毁了上一轮播放器实例？

## 后端

- 路径是否规范化并校验？
- 本地媒体服务是否限制在用户选择的根目录内？
- 分段请求是否仍然正常？
- 子进程启动是否避免命令注入风险？
- mpv 进程是否能清理？
- 后端错误是否能被前端理解？

## 播放

- `.ts` 播放仍能启动。
- `.ts` 拖动仍可用。
- `.mp4` 播放仍可用。
- 如果已实现 mpv，mpv 兜底仍可用。
- 不支持格式有清楚反馈。
- 如果涉及进度，播放进度保存仍可用。

## 开源

- 新依赖已写入 `THIRD_PARTY_NOTICES.md`。
- 新打包二进制已记录版本、来源和许可证。
- 许可证兼容性已检查。
- EasyPlayer.js 未在许可证确认前进入安装包。

## 文档

- `doc/` 下相关文档已更新。
- 如果用户行为变化，根目录 `README.md` 已更新。
- 如果用户可见行为变化，`CHANGELOG.md` 已更新。
- 如果发布流程变化，发布检查清单已更新。

## 验证命令

项目实现后的预期命令：

```powershell
pnpm run typecheck
pnpm run build
pnpm run tauri:build
```
