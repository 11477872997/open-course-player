# 安全策略

## 支持版本

当前还没有公开发布版本。

## 漏洞报告

如果仓库托管在 GitHub，请优先使用 private security advisory。否则通过项目公开的维护者联系方式报告。

请不要直接公开披露以下问题：

- 可以读取用户选择根目录之外的文件。
- 本地媒体服务暴露到局域网。
- 媒体路径导致命令注入。
- 可以任意控制 mpv 进程通信。
- 静默上传本地文件信息。

## 安全边界

- 本地媒体服务必须绑定 `127.0.0.1`。
- 后端读取或服务文件前必须校验路径。
- 应用默认不做全盘扫描。
- 子进程调用不能用原始字符串拼接 shell 命令。
